# Inclusion/Exclusion Criteria Checklist

## Quick Screening Guide

### INCLUDE if paper meets ALL of:
- [ ] Published 2015-2024
- [ ] English language
- [ ] Available full text (or can be retrieved)

### AND meets AT LEAST ONE of:
- [ ] Discusses anthropomorphization of AI/ML systems
- [ ] Studies human-like features in chatbots/LLMs
- [ ] Examines psychological effects of anthropomorphized AI
- [ ] Addresses ethical implications of human-like AI
- [ ] Focuses on vulnerable populations and AI
- [ ] Discusses trust/manipulation in AI systems
- [ ] Covers AI regulation/governance related to anthropomorphism

### EXCLUDE if:
- [ ] Physical robots only (no AI anthropomorphism component)
- [ ] Anthropomorphization of non-AI entities only
- [ ] Pure technical ML paper (no human interaction)
- [ ] Opinion piece without systematic analysis
- [ ] Published before 2015
- [ ] Non-English
- [ ] Full text unavailable

## Inter-rater Reliability Protocol

### First Pass (Strict Criteria)
- Apply all criteria conservatively
- When in doubt, exclude
- Focus on explicit mentions of key terms

### Second Pass (Lenient Criteria)
- Apply criteria more broadly
- Include borderline cases
- Consider implicit relevance

### Conflict Resolution
- If passes disagree, apply third review
- Use middle-ground interpretation
- Document reasoning for final decision

## Quality Gate
- Target Cohen's Kappa: κ > 0.6
- If κ < 0.6: Refine criteria and re-screen
- Document final κ value in results
